# Language Translation Tool

## Project Title
AI powered multilingual language translation tool

## Objective
A simple application that translates text from one language to another using a translation API. The application allows users to select source and target languages, enter text, and receive the translated output instantly.

## Features
- User-friendly interface
- Select source language
- Select target language
- Translate text instantly
- Copy translated text
- Clear input and output fields
- Error handling for invalid input

## Working Procedure
1. Open the application by running `python main.py`.
2. Enter the text to translate.
3. Select the source language.
4. Select the target language.
5. Click the Translate button.
6. The translated text appears in the output box.
7. Copy the translated text if required.
